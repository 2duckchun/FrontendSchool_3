float : 띄우다, (요소를 띄워서 좌 또는 우로 정렬을 하는 속성을 가짐) / 세빛둥둥섬 같은.. floating island
왼쪽, 오른쪽으로 정렬이 되는 특성 : 블록 박스 요소를 정렬하는 가장 기본적인 방식으로 사용됨.
float : left
float : right

이상하고 재미있는 float 속성 : 자식한테 float 속성을 주게되면
자식 요소들은 붕 띄워진 상태이기 때문에 부모 요소가 자식 박스들의 존재를 인식하지 못하게 됨.
 - 부모 요소에 overflow 속성을 추가 (overflow는 BFC(Block formatting context)를 생성) - BFC는
   float 속성이 적용된 요소를 컨테이너가 인식하도록 만들어 줌.

BFC
 - 웹 페이지 화면에 CSS를 랜더링하는 과정의 한 부분
 - block 레벨 요소들이나 float 된 요소들이 서로 상호작용하여 화면에 보여지게 되는 방법(block formatting)을 결정하는 구역(context)

BFC 요소가 생성될 때
html 요소를 사용했을 때
float : left, right
overflow에 visible 제외
display : table-cell, inline-block, flow-root
position:absolute, fixed 등등

BFC의 작용
내부, 외부 float 해제
