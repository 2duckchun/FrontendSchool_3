웹킷 기반 브라우저?

웹킷은 브라우저가 HTML, CSS를 화면에 그려줄 때 사용하는 렌더링 엔진. 크롬, 안드로이드, 사파리, ios 기반 파이어폭스 등 많은 브라우저들이 사용했음.

밴더 프리픽스는 사라지고 있는 중. (현재의 크롬, 안드로이드, 오페라, 엣지 브라우저는 Blink 엔진으로 전환 : blink 엔진은 vendor-prefix가 없음)

레거시 브라우저를 위해 벤더 프리픽스를 쓰긴 해야함.

이를 위해, autoprefixer 라이브러리를 통해 사용하면 됨.

VSC의 Extention에도 autoprefixer가 있음.
